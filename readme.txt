Render the activity of custom roles.

== Description ==

This plugin enhanced the functionality of the Activity Log plugin to show all the activities performed by users who have assigned custom roles.

== Installation ==

1. Upload plugin files to your plugins folder, or install using WordPress' built-in Add New Plugin installer
2. Activate the plugin
3. Go to the plugin page (under Dashboard > Activity Log)
